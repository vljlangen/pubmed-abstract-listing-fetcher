PubMed abstract listing (GitHub Actions)

Paste numbered references, one per line. This page prepares your run and links you to GitHub, where the same pubmed_abstracts.js logic runs on a server with a live log (so you can see “Processing 3 / 20 …” and know it is not stuck).

Why two places? A plain GitHub Pages site cannot securely start Actions or read live logs from your browser (no API token in the page, and GitHub’s API blocks normal web pages). So: prepare here → run on GitHub → watch the log there → download the zip from the finished run.

  • 1. Paste below (or load a sample from your repo’s references.txt elsewhere).
  • 2. Click Copy for GitHub, then Open GitHub Actions.
  • 3. On GitHub: Run workflow → paste into the references box (leave empty to use references.txt from the repo instead).
  • 4. Open the running job → expand Generate PubMed listing — the stream matches what you see locally in the terminal.
  • 5. When the job finishes: Artifacts → download pubmed-abstracts-output.zip (HTML + plain text + PDF).

Outputs. One zip contains pubmed_abstracts.html (rich layout), pubmed_abstracts.txt (stripped text), and pubmed_abstracts.pdf (simple printout from that text). Prefer the HTML for quality; use TXT/PDF for quick sharing or printing.
GitHub Pages: enable Pages from the /docs folder in repo Settings if this file is not yet served.
Respect NCBI policies and rate limits.
